# Hania Arshad — Portfolio

A personal portfolio website built with **Next.js** and **Tailwind CSS**, fully responsive for both **mobile and web**.

## Sections

- **Hero** — Profile image with blur-reveal on hover (desktop) / tap-to-view overlay (mobile), typewriter effect, call-to-action buttons, and social links
- **About** — Detailed intro with a second profile image, location, email, availability, and tech tags
- **Skills** — Categorized skills with proficiency bars and filter tabs
- **Projects** — Project cards with hover overlays (GitHub & live demo links), tech stack tags, and featured badges
- **Experience** — Timeline layout showing work history with role, company, period, location, type badge (Internship / Full-time), description bullets, and tech tags
- **Education** — Grid cards with institution, degree, GPA, achievements, and coursework
- **Services** — Service offerings with icons and descriptions
- **Contact** — Contact form with name, email, subject, message fields (email via API route); info sidebar with email, location, availability, and social links
- **Footer** — Navigation links, social icons, scroll-to-top button, copyright

## Features

- Dark / light theme toggle with localStorage persistence
- Custom pinkish-lilac cursor (desktop only)
- Scroll-triggered fade-in animations
- Typewriter title effect on hero
- Mobile hamburger menu with overlay
- Fully responsive layout (mobile-first)

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Nodemailer (contact form)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
